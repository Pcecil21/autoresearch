# AI Tool Usage Policy - Peter Cecil Financial Planning LLC
# Effective: April 2026

## NEVER put in any cloud AI tool:
- Client full names paired with financial data
- Social Security numbers, tax IDs
- Account numbers (brokerage, bank, IRA, 401k)
- Login credentials or passwords
- Client addresses paired with identifying info
- Specific dollar amounts tied to identified clients

## ALWAYS anonymize before pasting:
- Use Client A / Client B or first-name-only
- Round dollar amounts (1.2M IRA not 1,187,342.67)
- Remove account numbers - use Traditional IRA not IRA ending in 4829
- Strip custodian-specific identifiers

## ACCEPTABLE (already anonymized / non-PII):
- General portfolio allocations (60/40, 1.5M, married couple age 62)
- Tax scenarios with rounded numbers and no names
- Strategy questions, market research, fund analysis
- Code using sample/synthetic data
- Compliance policy drafting, templates, operational workflows

## Pre-Session Gut Check:
Could this data be traced back to a specific client if made public?
If YES - anonymize further before proceeding.
